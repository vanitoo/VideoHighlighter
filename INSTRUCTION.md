# 📹 Инструкция по использованию VideoHighlighter

## 🎯 Как нарезать шортсы из длинного видео

### Шаг 1: Подготовка

1. **Убедитесь, что FFmpeg установлен**
   - Скачайте с https://ffmpeg.org/download.html
   - Добавьте в PATH системы

2. **Запустите программу**
   ```bash
   python main.py
   ```

### Шаг 2: Добавление видео

1. Нажмите кнопку **"Add Videos"** (Добавить видео)
2. Выберите видеофайл(ы) на компьютере
3. Видео появится в списке вверху окна

### Шаг 3: Настройка параметров (опционально)

#### Вкладка "Basic Settings" (Основные настройки):

- **Scoring Points** — баллы за разные типы контента:
  - `Motion event points` (3) — баллы за движение в кадре
  - `Audio peak points` (0) — баллы за громкие звуки
  - `Object points` (1) — баллы за обнаруженные объекты
  - `Action points` (10) — баллы за действия (прыжки, бег и т.д.)

- **Duration & Cutting** — настройки длительности:
  - `Max highlight duration` — максимальная длительность хайлайта (по умолчанию 420 сек = 7 мин)
  - `Clip time` — длительность одного клипа (10 сек)
  - Если поставить `0`, включится авто-сегментация

- **Object detection** — какие объекты искать:
  - Введите через запятую: `person,car,ball`
  - Или нажмите **"Load Labels"** для выбора из списка

- **Action keywords** — какие действия искать:
  - Введите: `jumping,running,dancing`
  - Или нажмите **"Load Labels"** для выбора из 400 классов

#### Вкладка "Transcript & Subtitles" (Транскрипт и субтитры):

- Включите **"Enable transcript processing"** для анализа речи
- Выберите язык источника
- Добавьте **search keywords** для поиска определённых слов

#### Вкладка "Advanced" (Дополнительно):

- **Frame skip** — пропуск кадров для ускорения
- **YOLO model size** — размер модели детекции (n = быстрая, x = точная)
- **Action backend** — бэкэнд для распознавания действий

### Шаг 4: Запуск обработки

1. Убедитесь, что видео добавлено в список
2. Нажмите кнопку **"Run Highlighter"** (зелёная кнопка внизу)
3. Дождитесь завершения обработки (смотрите лог внизу окна)
4. Результат сохранится как `*_highlight.mp4` рядом с исходным видео

### Шаг 5: Ручное редактирование (Timeline Viewer)

Если хотите подправить нарезку:

1. После обработки нажмите **"📊 Show Timeline Viewer"**
2. Откроется окно с визуализацией сигналов:
   - Зелёные полосы — движение
   - Синие — аудио пики
   - Жёлтые — объекты
   - Фиолетовые — действия

3. **Редактирование клипов:**
   - Выделите клип (кликните по нему)
   - Нажмите **Delete** для удаления
   - Перетаскивайте клипы для изменения порядка
   - Двойной клик — воспроизведение клипа

4. **Добавление клипов:**
   - Кликните в любом месте видео (Ctrl + клик для добавления)
   - Или перетащите сигналы из верхней панели в нижнюю

5. Нажмите **"X"** для закрытия и возврата к результату

## 💡 Советы

### Для шортсов/тик-токов:
1. Установите `Max highlight duration` = 60 (макс 60 сек)
2. `Clip time` = 10-15 сек
3. Включите `Motion event points` = 5
4. Включите `Audio peak points` = 3
5. Добавьте объекты: `person`
6. Добавьте действия: `jumping,dancing,singing`

### Для фильмов/сериалов:
1. Увеличьте `Max highlight duration` = 300-600
2. Включите `Action points` = 10
3. Добавьте действия: `fighting,running,talking`
4. Включите `Transcript` для поиска по диалогам

### Для спортивных видео:
1. `Motion event points` = 5
2. `Audio peak points` = 5
3. `Action points` = 15
4. Действия: `goal,score,catch,throw`

## ⚙️ Полезные функции

### Загрузка видео с сайтов:
1. Перейдите на вкладку **"Download"**
2. Введите URL страницы с видео
3. Укажите паттерн ссылок (обычно `/video/`)
4. Выберите папку сохранения
5. Нажмите **"🌐 Download Videos"**

### Автоматическая обработка после загрузки:
- Включите **"Process each video immediately after download"**
- Видео будет обработано автоматически после скачивания

### Избегание людей:
1. Включите **"Enable face recognition"** во вкладке "Avoid"
2. Нажмите **"🔍 Scan video for faces"**
3. Выберите лиц, которых нужно исключить
4. Они будут удалены из нарезок

## ❓ Решение проблем

### Программа не открывает видео:
- Убедитесь, что FFmpeg установлен и в PATH
- Проверьте, что видео в поддерживаемом формате (mp4, avi, mkv)

### Обработка зависает:
- Увеличьте `frame_skip` и `object_frame_skip`
- Используйте YOLO модель размера `n` (nano) вместо `x`
- Обрабатывайте только часть видео через "Processing Time Range"

### Нет звука в хайлайтах:
- Включите `audio_peak_points` > 0
- Проверьте, что видео содержит аудиодорожку

### Медленная работа:
- Отключите ненужные функции (транскрипт, действия)
- Используйте CPU для YOLO, если нет GPU
- Уменьшите `sample_rate` для действий

## 📁 Где находятся файлы?

- **Исходное видео:** там, где вы его выбрали
- **Хайлайт:** рядом с исходным видео, с суффиксом `_highlight.mp4`
- **Кэш:** папка `./cache/` в директории программы
- **Лог:** внизу главного окна

## 🔄 Обновление кэша

Если изменились параметры:
1. Нажмите **"Clear All"** в списке видео
2. Добавьте видео заново
3. Или подождите, пока кэш автоматически обновится
