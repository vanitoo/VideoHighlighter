## 🐛 Известные баги

- [ ] Google Translate может сломаться (unofficial API)
- [ ] OpenVINO экспорт требует PT-файла при первом запуске
- [ ] Большие видео (>1GB) могут вызывать OOM
- [ ] Face recognition работает медленно на CPU

## 💬 Заметки

### Архитектурные решения
- PySide6 выбран за Qt-экосистему и cross-platform
- Многопоточность через QThread (не asyncio)
- Кэш signature-based для инвалидации при изменении параметров

### Технический долг
- Много глобальных переменных в pipeline.py
- Дублирование кода в main.py и signal_timeline_viewer.py
- Нет type hints для некоторых функций
- Hardcoded пути ("./cache/", "./models/")

### Зависимости
- FFmpeg — внешняя зависимость
- torch — тяжёлая библиотека, долгая загрузка
- opencv-python — нет GPU-акселерации на macOS
